Feline Color Genetics: Consolidated Data Tables

Section 1: Feline Color Genetics Cheatsheet & Lexicon

1.1. Core Genetic Concepts

  • Orange (Sex-Linked Red): Located on the X-chromosome. It is epistatic, meaning it masks the expression of the black/brown gene.
  • Brown (Black vs. Chocolate): An autosomal gene determining if the base pigment is black (B) or brown/chocolate (b). This is a recessive trait.
  • Dilute (Dense vs. Dilute): An autosomal gene that lightens the base coat color. This is a recessive trait.

1.2. Gene & Allele Lexicon

1.4. Inheritance Pattern Cheatsheet

If you cross... You will get... You will NEVER get...
Dilute x Dilute (e.g., Blue x Blue) 100% Dilute kittens (Blue, Cream, Lilac). Dense kittens (Seal, Red, Chocolate).
Homozygous Chocolate x Homozygous Chocolate 100% Chocolate-based kittens (Chocolate, Lilac). Black-based kittens (Seal, Blue).
Non-Dilute x Non-Dilute If both parents carry dilute (`Dd`), you can get both Dense and Dilute kittens. -
Red Male (`XOY`) x Non-Orange Female (`XoXo`) 100% Non-Orange Males (`XoY`), 100% Tortie Females (`XOXo`). Red males, Red females, or Non-Orange females.
Non-Orange Male (`XoY`) x Red Female (`XOXO`) 100% Red Males (`XOY`), 100% Tortie Females (`XOXo`). Non-Orange males, Red females, or Non-Orange females.
Red Male (`XOY`) x Red Female (`XOXO`) 100% Red kittens. Kittens of any other color.
Any Sire x Tortie Dam (`XOXo`) Both Orange and Non-Orange males; both Non-Orange and Tortie females. -
Any Sire x Non-Orange Dam (`XoXo`) No Tortie females. Tortie females.
Two parents that are NOT chocolate-based (`BB` or `Bb`) If both parents carry chocolate (`Bb`), you can get both black-based and chocolate-based kittens. -
